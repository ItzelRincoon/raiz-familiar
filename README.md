# Raíz Familiar 🌳

Aplicación web responsiva (enfoque PWA) para que las familias mexicanas preserven, organicen y compartan su historia familiar de forma privada.

Proyecto desarrollado como parte de la asignatura **Desarrollo Web Integral** — Universidad Tecnológica de Puebla.

## Módulos del sistema

- **Onboarding** — Registro e inicio de sesión.
- **Home Dashboard** — Panel principal de la familia.
- **Baúl de Recuerdos** — Galería de fotos, audios y textos.
- **Árbol Familiar** — Representación visual del parentesco.
- **Eventos** — Calendario de fechas importantes.
- **Recetas (Cookbook)** — Recetario familiar con su historia.
- **Cápsulas del tiempo** — Mensajes que se abren en fecha futura.
- **Tablero SOS** — Aviso rápido entre familiares en emergencias.
- **Invitaciones** — Creación de grupo familiar y códigos de invitación.
- **Configuración** — Ajustes de cuenta y del grupo familiar.

## Tecnologías

- React + TypeScript + Vite
- Tailwind CSS
- Firebase (Authentication, Firestore, Storage) — capa de backend
- Metodología: Scrum

## Instalación

```bash
npm install
npm run dev
```

## Flujo de trabajo (Git)

Este repositorio se desarrolló siguiendo un flujo de ramas por módulo (feature branches), con integración vía Pull Request hacia `main`. Ver documentación en `/docs` para el detalle de comandos usados.
